ESERCIZIO BTS_1.1

1. A cosa serve bootstrap?

Bootstrap è un framework front-end gratuito per uno sviluppo Web più rapido e semplice
Bootstrap include modelli di progettazione basati su HTML e CSS per tipografia, moduli, pulsanti, tabelle, navigazione, modali, caroselli di immagini e molti altri, nonché plug-in JavaScript opzionali
Bootstrap ti dà anche la possibilità di creare facilmente progetti reattivi

2. Cosa devi importare per creare una pagina web con bootstrap

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css">

3. Cos’è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?

Una Content Delivery Network è una rete di server in diversi centri di elaborazione dati in tutto il mondo.Il vantaggio è che la maggior parte dei CDN si assicurerà che una volta che un utente richiede un file da esso, verrà servito dal server più vicino a loro, il che porta anche a tempi di caricamento più rapidi.

4. A cosa serve questo attributo del tag meta?

Rende le pagine web concepite per il desktop senza alcun problema.
Stiamo dicendo al browser di impostare la larghezza (width) della viewport in base alla larghezza dello schermo del dispositivo (device-width) e La proprietà initial-scale imposta il fattore di zoom iniziale, relativo al momento in cui la pagina viene caricata.

5. Che differenza c’è tra Container e Container Fluid (prova il container fluid creando una pagina chiamata BTS_1.1.1.html)

1. La .containerclasse fornisce un contenitore responsive a larghezza fissa
2. La .container-fluidclasse fornisce un contenitore a larghezza intera , che copre l'intera larghezza della finestra
